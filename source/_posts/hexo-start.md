### 安装全局hexo
``` bash
$ npm i hexo -g
```

### 安装依赖
``` bash
$ npm i
```

### Run server
``` bash
$ hexo s
```

### 生成新目录
``` bash
$ hexo new page 'xxx'
```

### 生成新页面
``` bash 
$ hexo new 'xxx'
```

### 生成静态文件
``` bash
$ hexo g
```

### 发布至远端站点
``` bash
$ hexo d
```
